KASH Component Strategy

Spécifications Détaillées - Composants MVP & Personnalisés

🏗️ Strategy Overview

Design System Foundation: MUI + Tailwind UI avec tokens personnalisés KASH

Approach: 70% composants MUI standards + 30% composants custom KASH

Goal: Cohérence visuelle entre Lina (étudiant) et M. Karim (conseiller) avec variants adaptés

📊 KASHScoreCard

Dashboard central étudiant - 4 scores KASH

Purpose

Dashboard central affichant les 4 scores KASH (Knowledge, Abilities, Skills, Intelligence) avec indicateur global de progression

Content

• 4 scores numériques (0-100%)
• Progress bars visuelles
• Score global pondéré
• Timestamp dernière mise à jour
• Badge niveau progression

Actions

• Click → détails dimension spécifique
• Hover → tooltip explication simple
• Long press → mode comparaison historique
• Share → export profil simplifié

States

Default
Hover
Loading
Error
Success

Variants

• Student: Compact, gamifié, encourageant
• Admin: Dense, détaillé, comparatif
• Mobile: Vertical, swipe-friendly

⭕ ProgressRing

Visualisation progression gamifiée

Purpose

Visualisation circulaire de progression avec gamification pour engagement et motivation continue

Content

• Pourcentage progression (0-100%)
• Badge/achievement courant
• Prochaine étape/milestone
• Streak count (jours consécutifs)
• Micro-animations de completion

Actions

• Click → détails progression
• Hover → prochaines actions suggérées
• Double-click → célébration achievement
• Swipe → navigation entre étapes

States

Empty
Progress
Complete
Stalled
Celebration

Variants

• Quiz: Progression questions
• Daily: Check-in quotidien
• Weekly: Récap hebdomadaire
• Overall: Parcours complet
[Mockup: ProgressRing circulaire avec badge et streak]

🎯 RecommendationCard

Recommandations actionnables métiers/formations

Purpose

Présenter recommandations personnalisées (métiers, formations) avec actions directes et contexte KASH

Content

• Titre recommandation (métier/formation)
• Score de compatibilité (0-100%)
• Écarts compétences addressés
• Durée/coût estimé
• Actions suggérées (apply, learn more)
• Tags (remote, local, level)

Actions

• Click → détails complets
• Add to Plan → ajouter plan d'action
• Bookmark → sauvegarder pour plus tard
• Share → envoyer à conseiller
• Dismiss → pas intéressé(e)

States

New
Viewed
Saved
Dismissed
Applied

Variants

• Career: Métiers et professions
• Training: Formations et cours
• Skill: Compétences spécifiques
• Compact: List view mobile
[Mockup: RecommendationCard avec score + actions]

🔍 SHAPExplanation

Transparence scoring ML simplifiée

Purpose

Rendre les recommandations ML transparentes et compréhensibles pour construire la confiance des utilisateurs

Content

• Graphique importance features (barres)
• Top 3 facteurs influents
• Explication en langage simple
• Impact sur score final
• Toggle detail/résumé

Actions

• Hover → détail facteur spécifique
• Click → explication complète
• Toggle → simple/détaillé
• Export → sauvegarder explication
• Learn more → lien documentation

States

Summary
Detailed
Loading
Interactive
Exported

Variants

• Student: Simple, visuel, encourageant
• Admin: Détaillé, technique, exportable
• Tooltip: Mini version inline
• Modal: Vue complète

🚨 AlertDashboard

Vue priorisée étudiants pour conseillers

Purpose

Interface conseiller pour identifier rapidement les étudiants nécessitant une attention prioritaire

Content

• Liste étudiants avec alertes
• Niveau priorité (high/medium/low)
• Raison alerte (score, inactivité, etc.)
• Actions suggérées
• Filtres rapides
• Timeline interventions

Actions

• Click étudiant → fiche complète
• Assign action → plan intervention
• Mark resolved → archive alerte
• Bulk actions → traiter multiples
• Export → rapport interventions

States

Critical
Warning
Info
Resolved
Archived

Variants

• List: Vue détaillée complète
• Compact: Vue rapide priorités
• Timeline: Chronologique interventions
• Analytics: Vue statistique
[Mockup: AlertDashboard avec liste priorités]

📈 GapVisualization

Identifier écarts compétences comme opportunités

Purpose

Visualiser les écarts entre compétences actuelles et requises pour les métiers cibles de manière positive

Content

• Graphique radar ou barres comparatives
• Compétences actuelles vs requises
• Scores de compatibilité
• Zones d'opportunité highlightées
• Recommandations associées

Actions

• Hover compétence → détails écart
• Click zone → recommandations ciblées
• Toggle view → radar/barres
• Compare → vs autres métiers
• Export → rapport compétences

States

Radar
Bars
Comparison
Detailed
Exported

Variants

• Radar: Vue globale 360°
• Bars: Comparaison détaillée
• Gap: Focus écarts seulement
• Opportunity: Vue positive


Nom du produit :
KASH Career Intelligence Platform

Problème :
Les étudiants en fin de cycle universitaire ont du mal à identifier leur métier idéal. Les solutions actuelles sont mono-dimensionnelles et ne fournissent ni prédiction fiable ni explication sur leur potentiel global.

Solution :
Une plateforme d’IA évaluant toutes les dimensions KASH (Knowledge, Abilities, Skills, Habits) avec 4 méthodes distinctes validées scientifiquement, mappant le profil aux métiers, prédisant l’alignement et générant des recommandations explicables.

Proposition de valeur :
Seul système d’IA 4D basé sur KASH qui prédit la réussite professionnelle avec transparence et justification scientifique.

Segment cible (MVP) :
Étudiants 18–25 ans en fin de cycle universitaire
Écoles d’ingénieurs, business schools, grandes universités

Modèle économique (MVP) :
B2B2C via partenariats établissements (€/étudiant/an)
Freemium B2C pour rapports premium

Scope MVP :

Dimension KASH	Méthode de mesure
Knowledge	Analyse CV via NLP, mapping ESCO/O*NET, quiz adaptatifs
Abilities	Tests cognitifs, scénarios analytiques, résolution de problèmes
Skills	Mini-projets, challenges techniques, analyse code / portfolio
Habits	Tracking comportemental, fréquence d’apprentissage, analyse GitHub / portfolio

Fonctionnalités MVP prioritaires :

Évaluation KASH multidimensionnelle (4 méthodes distinctes)

Tableau de bord 4D simple avec évolution temporelle

Mapping KASH ↔ 50 métiers de référence

Score de compatibilité KASH–métier

Prédiction initiale d’alignement avec IA explicable simplifiée

Fonctionnalités post-MVP / Phase 2 :

Marketplace mentorat KASH

Réseau social collaboratif

Badges / micro-certifications

Simulations de parcours alternatifs

API avancée pour entreprises

Objectifs MVP (6 mois) :

Pilote dans 2 écoles, 100 étudiants

Adoption étudiants >60%

Accuracy prédictive >70%

Satisfaction établissements >4/5

Retention 3 mois >80%

Vision scientifique et technique :

Fusion multimodale pour Attitude (texte, voix, visage)
